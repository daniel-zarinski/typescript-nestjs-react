## Installation

```zsh
$ yarn install
```

## Running

```bash
# Start & Watch for API & lib changes (Port 8080)
$ yarn start

# Start React by running webpack dev server (Port 3000)
$ yarn start:web
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
